This is my project for my Structure for Computer Systems laboratory, it is an Android application that provides the user with the specifications of their device, and allows the user to start any test they choose from a selcetion of predefined tests. 
These tests aim to assess the performance of the device in the follwing secenarios:
  - Single and multi-core CPU performance
  - GPU performance
  - Memory bandwidth
  - Storage throughput

The application is developed inside the Android Studio API, the language used for the UI is Kotlin, and the performance tests are written in C, to better illustrate the performance of the device. 
A more detailed overview is availabele inside the pdf documentation of the project, which is attached inside this repository.
